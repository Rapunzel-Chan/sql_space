SQL Космический Тренажер

Изучайте SQL, покоряя цифровую вселенную

Основы SELECT

Научитесь извлекать данные из таблиц с помощью базовых запросов SELECT.

SELECT * FROM users WHERE age > 25;
Начать обучение

JOIN и отношения

Освойте соединение таблиц и работу с реляционными данными.

SELECT u.name, o.total FROM users u JOIN orders o ON u.id = o.user_id;
Изучить JOIN

Фильтрация данных

Научитесь точно фильтровать данные с помощью WHERE и HAVING.

SELECT product, COUNT(*) FROM sales GROUP BY product HAVING COUNT(*) > 10;
Фильтровать

Сортировка и агрегация

Используйте ORDER BY и агрегатные функции для анализа данных.

SELECT department, AVG(salary) FROM employees GROUP BY department ORDER BY AVG(salary) DESC;
Анализировать

Изменение данных

Научитесь добавлять, изменять и удалять данные с помощью INSERT, UPDATE, DELETE.

UPDATE products SET price = price * 1.1 WHERE category = 'electronics';
Модифицировать

Подзапросы

Освойте мощь подзапросов для сложных выборок данных.

SELECT name FROM employees WHERE salary > (SELECT AVG(salary) FROM employees);
Использовать подзапросы
